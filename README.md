## 🌟 What is SHAKTI.AI?

SHAKTI.AI helps home-based workers, SHG members, domestic workers, and rural women get their skills recognized, certified, and monetized — in their own language, by voice.

## ✨ Features
- 🌐 Multilingual + Voice-First (9 Indian languages)
- 🧠 AI Skill Assessment → Certified Course Recommendations
- 📷 AI Product Quality Checker (Gemini Vision)
- 🛒 Direct Marketplace — no middlemen
- 🏦 SHG Finance Tracker via voice
- 📚 Built-in Learning Platform with Certifications
- 🎙️ Voice Assistant "Shakti Didi" on every page

## 🛠️ Tech Stack
React • Tailwind CSS • Firebase • Gemini API • Vercel
